## API Features

<Columns cols={2}>
  <Card icon="cloud-arrow-up" href="/api-reference/introduction" title="Upload media">
    Upload images and videos via `POST /upload`. Up to 100 MB free, 5 GB with Plus.
  </Card>
  <Card icon="list" href="/api-reference/introduction" title="Manage uploads">
    List and delete your uploads programmatically with `GET /uploads` and `DELETE /deleteUpload`.
  </Card>
  <Card icon="user" href="/api-reference/introduction" title="User data">
    Fetch profile fields, badge metadata, and optional public profile uploads with `GET /users/{id}`.
  </Card>
  <Card icon="compass" href="/api-reference/introduction" title="Discover feed">
    Access public uploads for the discover feed with `GET /discover`.
  </Card>
</Columns>
